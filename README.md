**Quicklist** é um desafio prático desenvolvido durante a formação Full-Stack da **Rocketseat**. O projeto consiste em uma aplicação simples para gerenciar a lista de compras da semana, permitindo adicionar novos itens, marcar como concluídos e remover da lista quando não forem mais necessários.

---

## 📸 Visualização do Projeto:

<img alt="preview" src="/Thumbnail (5).png" />

---

## 🚀 Funcionalidades:

- Adicionar novos itens à lista de compras
- Marcar itens como concluídos
- Remover itens da lista
- Feedback visual para ações realizadas
- Interface responsiva (desktop e mobile)
- Armazenamento local dos dados

---

## 🛠️ Tecnologias utilizadas:

- HTML5  
- CSS3  
- JavaScript (ES6+)
