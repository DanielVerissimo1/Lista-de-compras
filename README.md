**Quicklist** é um desafio prático desenvolvido durante a formação Full-Stack da **Rocketseat**. O projeto consiste em uma aplicação simples para gerenciar a lista de compras da semana, permitindo adicionar novos itens, marcar como concluídos e remover da lista quando não forem mais necessários.

## 🚀 Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| ➕ **Adicionar itens** | Insira novos produtos na sua lista de compras |
| ✅ **Marcar concluído** | Dê check nos itens já comprados |
| 🗑️ **Remover itens** | Exclua itens que não são mais necessários |
| 💡 **Feedback visual** | Animações e notificações para cada ação |
| 📱 **Responsivo** | Funciona perfeitamente no desktop e mobile |
| 💾 **Persistência** | Sua lista é salva automaticamente no navegador |
